
## 3.3 pre 태그 (Preformatted text)

pre 태그 내부에 있는 컨텐츠는 작성한 그대로 화면에 보이게 된다.

```
<pre>
나는 프리하다.

나는 마크업에 작성하는 그래도 화면에 보여진다.
공       백이나   


줄바꿈도 마찬가지다.
</pre>
```

[jsfiddle](https://jsfiddle.net/cmf70q6w/)

## 3.4 hr 태그

hr태그는 수평줄을 추가한다.

```
<!DOCTYPE html>
<html>
  <body>
    <h1>다음</h1>
    <p>다음에</p>
    <hr>
    <h1>네이버</h1>
    <p>네버</p>
  </body>
</html>
```

[jsfiddle](https://jsfiddle.net/L5g7f389/)
